Bible reader

By: Christian Coder
A free and open source Bible reader. A Progressive Web App available to download from this web page. This app is not found in the IOS or Android app stores.
  • Always free
  • Open source available on GitHub
  • No ads
  • No login or sign-up
  • No in-app offers
  • No credit cards
  • No user statistical tracking
How to Install a PWA on iPhone:
  • Open this web page using the Safari browser (click here).
  • Tap the "Share" button (a square with an arrow pointing out of it).
  • Select "Add to Home Screen" from the menu.
  • Tap "Add" in the top right corner.
